# Tic Tac Toe 
### Cli interface 

To start the game use `lein run`

Possible positions:

| 1   | 2   | 3   |
|-----|-----|-----|
 | 4   | 5   | 6   |
| 7   | 8   | 9   | 

=> lein run: 

Enter a number between 1-9 to make a move

Enter h to view position board

Enter q to quit
